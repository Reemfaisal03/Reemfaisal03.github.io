User Manual

  1. How to Use the NeuroCode System

    Launch the System:

    • Wear the VR headset.
    • Start the NeuroCode platform on the connected device.

  2. Login

    • Enter your login credentials (if available).
    • Or choose "Guest Mode" for limited access.

  3. Select a Lesson

    From the main menu, browse the list of interactive programming lessons:

    • Choose a topic (e.g., loops, conditionals, data structures).

  4. Start the Interactive Learning

    • You will enter a 3D environment representing the selected programming concept.
    • Listen to the AI Assistant explaining the topic step-by-step using Text-to-Speech.

  5. Practice Coding

    • Use the tools in the VR environment to write and test code.
    • Run the code and receive immediate feedback from the AI Assistant.

  6. Ask for Help

    At any time, you can ask the assistant (via voice or text) for explanations or code corrections.

  7. End the Session

    After completing the lesson, you can save your progress or safely exit the platform.
